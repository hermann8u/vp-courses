# Listes de bundles

Vous trouverez ici une liste de quelques bundles Symfony intéressant à connaître/utiliser

  • Symfony
  • Doctrine
  • FriendsOfSymfony
  • Autre

# Symfony

  • symfony/framework-bundle (opens new window) : Le bundle contenant le framework lui-même

  • symfony/twig-bundle (opens new window) : Intégration de Twig dans Symfony (Ajout de méthodes et de la configuration)

  • symfony/webpack-encore-bundle (opens new window) : Intégration de Webpack pour la gestion des assets avec une simplification de la syntaxe de configuration.

  • symfony/maker-bundle (opens new window) : Permet de générer du code depuis la console (disponible depuis la version 4 de Symfony).

  • sensio/framework-extra-bundle (opens new window) : Gestion des annotations pour les contrôleurs (Routing, View, Cache, Security, ParamConverter, ...). Il est donc nécessaire pour pouvoir utiliser les annotations comme type de routing.

# Doctrine

  • doctrine/doctrine-bundle (opens new window) : Le bundle intégrant l'ORM Doctrine à Symfony.
  • doctrine/doctrine-fixtures-bundle (opens new window) : Permet de gérer des fixtures dans un projet Symfony utilisant Doctrine.
  • doctrine/doctrine-migrations-bundle (opens new window) : Intègre le système de migrations de base de données à l'ORM Doctrine.

# FriendsOfSymfony

  • friendsofsymfony/rest-bundle (opens new window) : Fournit différents outils pour faciliter le développement d'une api REST avec Symfony.
  • friendsofsymfony/elastica-bundle (opens new window) : Permet d'utiliser Elacticsearch avec son wrapper PHP Elastica dans une application Symfony.
  • friendsofsymfony/jsrouting-bundle (opens new window) : Permet d'exposer certaines routes de notre application au client pour les utilser en Javascript. Pratique pour faire des appels AJAX !

# Autre

  • stof/doctrine-extensions-bundle (opens new window) : Ajoute des annotations pour réaliser des tâches courantes sur vos entités Doctrine (Génération de slug, Gestion des createdAt/updatedAt, ...).
  • lexik/jwt-authentication-bundle (opens new window) : Gestion de Json Web Token pour gérer l'authentification. Principalement si vous développez une API.
  • easycorp/easyadmin-bundle (opens new window) : Génération de tout un back-office en se basant sur vos entités Doctrine.
  • api-platform/api-pack (opens new window) : API Platform permet de générer toute une API en se basant sur vos entités Doctrine.
  • liip/imagine-bundle (opens new window) : Redimensionnement et mis en cache d'image pour l'affichage.
  • babdev/pagerfanta-bundle (opens new window) : Fournit différents outils pour gérer la pagination.
  • whiteoctober/breadcrumbs-bundle (opens new window) : Fournit différents outils pour générer un "breadcrumb" (ou fil d'ariane en français) sur votre application.
